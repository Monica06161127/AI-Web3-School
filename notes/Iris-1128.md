# 2026-05-22
<!-- DAILY_CHECKIN_2026-05-22_START -->
一、本周最大收获：重新理解“AI × Web3 的本质”

之前我一直以为 AI + Web3 就是用 AI 写合约、用区块链存数据。但听完5.20的「Web3 运行原理」直播后，我的认知被刷新了：

\> **AI × Web3 的核心不是技术叠加，而是让 AI 拥有“经济主体”身份。**

当 AI Agent 从“回答问题”走向“执行任务”（比如租算力、买数据、支付 API 费用、与其他 Agent 协作），它必须有一个可被授权、能付款、可追责的链上账户。

Web3 提供的正是这种去中心化的经济基础设施——不需要中介，不需要银行，只有公私钥和智能合约。

二、两个让我卡了很久的问题（以及现在的理解）

1.**签名和交易到底什么关系？**

以前以为签名就是“确认按钮”。现在明白：签名只是对一段消息的加密证明，它本身不上链；只有当你把签名附在一个交易里广播出去，矿工打包后，才算完成链上执行。

→这解释了为什么 AI Agent 不能直接拿着私钥签名就完事，它还需要 Gas、Nonce 和广播机制。

2.**测试网交互的意义是什么？**

我原以为就是“练手”。但实际走完一遍（领水→转账→部署合约→Etherscan 查）才体会到：测试网让你完整体验\*\*私钥控制权、Gas 成本、交易确认时间、失败重试\*\*这些主网完全一致的约束。没有测试网，不敢上主网。

三、我做的两个最小验证结果

\- **AI 工具实践**：用 Cursor 的 Chat 功能写了一个简单的 Solidity 存证合约，AI 生成了基本框架，我手工补充了事件和 modifier。验证了 AI Coding 能提效，但不能替代人工审查（AI 一开始忘了加权限控制）。

\- **AI × Web3 交叉实验**：让 ChatGPT 输出一道关于“私钥泄露后果”的单选题，我选答案后，在 Remix 上部署了一个记录正确/错误的合约，把我的答案作为参数调用。验证了“AI 输出 → 人工复核 → 链上不可篡改记录”的可行性。

一、核心概念速览

\-**账户/地址/钱包**：地址是公开标识（如 `0x...`）；钱包是管理私钥的工具；私钥/助记词是账户控制权，\*\*绝对不能泄露\*\*。

\-**签名 vs 交易**：签名是私钥对消息的授权证明；交易是附带签名的链上请求，需支付 Gas。

\-**Gas**：交易执行的手续费（ETH），防止网络滥用。L2 费用更低。

\-**智能合约**：链上公开代码，部署后多数不可更改；读取免费，写入需 Gas。

\-**主网 vs 测试网**：主网用真实资产；测试网（如 Sepolia）用免费 ETH，学习和实验先用测试网。

\-**区块浏览器**：查询交易、合约、地址状态（如 Etherscan），可查看交易哈希、Gas、区块高度等。

**学习笔记–2026-05-20**

一、AI 基础概念理解

\- **LLM（大语言模型）**：基于海量文本训练的深度学习模型，能理解和生成自然语言，如 GPT、Llama。

\- **Prompt（提示词）**：用户输入给 LLM 的指令，设计好的 prompt 能显著提升回答质量。

\- **Workflow（工作流）**：将多个步骤（如数据获取、模型调用、结果处理）串联成固定流程。

\- **Agent（智能体）**：能自主规划、调用工具、执行多步任务的 AI 程序。

\- **Tool Use（工具使用）**：LLM 或 Agent 调用外部 API、数据库等完成具体操作。

\- **AI Coding**：利用 LLM 辅助编写、调试、解释代码，提升开发效率。

二、Web3 链上操作链

一次完整的链上操作包括以下环节：

1.**账户**：由公钥派生出的地址，分 EOA（外部账户）和合约账户。

2.**钱包**：管理私钥/助记词的工具，用于签名和发起交易。

3.**签名**：用私钥对交易或消息进行加密签署，证明所有权。

4.**交易**：包含发送方、接收方、金额、数据等的链上请求。

5.**Gas**：交易执行的手续费，以 ETH 计价，防止网络滥用。

6.**合约**：部署在链上的可执行代码，定义业务逻辑。

7.**测试网**：模拟主网的环境，ETH 免费，用于开发和测试。

8.**区块浏览器**：查询交易、合约、地址状态的工具（如 Etherscan）。

**操作示例**：连接钱包→签署交易→支付 Gas→合约执行→浏览器确认。

三、最小交叉实验思路（AI × Web3）

让 LLM 生成一道判断题→用户选择答案→将答案提交到测试网合约→合约记录结果并发放积分→在区块浏览器查看交易记录。该流程体现了 AI 输出、人工复核、链上执行和验证记录的结合。
