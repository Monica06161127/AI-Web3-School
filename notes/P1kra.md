概念卡片写了两套。AI 那边纯新手，六个概念从 LLM 写到 RAG，每个都附了误区和边界——比如 LLM 不是搜索引擎，prompt 不是越长越好，Agent 不是自主 AI。Web3 这边八个概念从密码学写到安全，最后单独写了私钥、助记词、签名、授权的风控说明。  
  
晚上先参加 Co-learning，主要是答疑 Hermes 配置的常见卡点，大部分同学在登录 GitHub 和接 API 上花了时间。我是白天跑通的，比较顺。  
  
然后听了「AI 时代的 Web3 架构能力」。最大的收获就一句话：人与 AI 的协作需要分工——人设计方案和验收成果，AI 优化方案并执行。不是让 AI 替你决定，是你定方向、说标准，AI 帮你跑通和打磨。
