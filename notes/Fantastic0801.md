# 大模型基础

## 一、核心起源：函数与两大 AI 思想

1.  **函数本质**：是一种数据变换规则，输入数据经过变换得到目标结果。
    
2.  **符号主义**：早期人工智能思路，用**精准固定函数**描述所有事物关系，缺点：现实多数场景无法找到完美精确函数。
    
3.  **联结主义**：放弃精准拟合所有数据点，寻找**近似最优函数**，贴近真实结果即可，是神经网络核心思想。
    

## 二、激活函数：实现非线性变换

1.  **局限**：简单线性函数只能拟合直线，无法适配曲线类复杂数据关系。
    
2.  **解决方案**：在线性运算外嵌套**非线性函数**（平方、正弦、指数等），这类函数统称为**激活函数**。
    
3.  **作用**：打破线性局限，让模型具备拟合非线性数据的能力。
    

## 三、神经网络基础构成

1.  **多层嵌套原理**：多输入 + 多层线性变换 + 多层激活函数层层嵌套，理论可逼近任意连续函数。
    
2.  **神经元**：模型基础最小单元，图示中小圆圈即为神经元。
    
3.  **神经网络**：大量神经元相互连接形成网状结构。
    
4.  **网络层级划分**
    
    -   输入层：接收原始数据
        
    -   隐藏层：输入与输出中间层，核心作用**提取复杂数据特征**
        
    -   输出层：输出最终预测结果
        
5.  **前向传播**：数据从输入层→隐藏层→输出层，从左向右逐层计算传递信号的过程。
    
6.  **训练核心目标**：不断更新权重`w`、偏置`b`两大核心参数，拟合真实数据规律。
    

## 四、神经网络参数求解

### 1\. 损失函数

-   定义：计算**模型预测值**与**真实值**之间误差的函数。
    
-   常用类型：**均方误差**，对误差取平方后求平均值，直观反映整体拟合效果。
    
-   核心目标：训练全程不断缩小损失函数数值。
    

### 2\. 梯度下降

-   适用场景：复杂神经网络无法用数学公式直接求解最优`w、b`。
    
-   核心逻辑：根据损失函数变化方向调整参数，损失变大则反向调整参数。
    
-   梯度：损失函数对所有参数的偏导数组合向量。
    
-   学习率：控制参数每次调整的幅度大小（超参数）。
    
-   调整规则：所有参数朝着**梯度反方向**迭代更新。
    

### 3\. 反向传播

-   原理：依托**链式求导法则**，从输出层反向向左逐层计算损失函数对每一层参数的偏导数。
    
-   作用：快速求出所有神经元权重、偏置的梯度，配合梯度下降完成参数更新。
    

### 4\. 单次完整训练流程

前向传播计算预测结果 → 计算损失误差 → 反向传播求梯度 → 梯度下降更新参数，多轮迭代至损失值足够小。

## 五、模型训练常见问题：过拟合

1.  **过拟合定义**：模型过于复杂，学习到训练数据中的噪声、随机波动，**泛化能力极差**，新数据预测效果差。
    
2.  **泛化能力**：模型对从未见过的新数据的预测适配能力。
    
3.  主流解决办法
    
    1.  简化模型结构，减少神经元与网络层数
        
    2.  扩充训练数据集，现有数据可做数据增广（图像旋转、裁剪、镜像等）
        
    3.  早停训练，提前终止迭代
        
    4.  **正则化**：在损失函数添加参数惩罚项，限制参数过度增大
        
    5.  **Dropout**：训练时随机丢弃部分神经元，避免模型依赖单一参数
        

## 六、矩阵简化表达

1.  用矩阵运算简化繁杂函数公式，统一标注网络层`a[0]、a[1]、a[2]`。
    
2.  优势：适配 GPU 并行计算，大幅提升神经网络训练、推理速度。
    

## 七、主流经典神经网络架构

### 1\. 卷积神经网络 CNN（专攻图像领域）

1.  **全连接层缺陷**：参数数量庞大，无法捕捉图像像素位置、局部特征。
    
2.  **卷积运算**：利用固定大小卷积核遍历图像，提取边缘、轮廓、纹理等局部特征。
    
3.  **卷积核**：模型可自主训练学习的特征提取矩阵，区别于传统固定图像处理卷积核。
    
4.  **池化层**：降低数据维度，精简计算量，保留核心图像特征。
    
5.  整体结构：卷积层 + 池化层 + 全连接层组合，擅长图像分类、图像识别。
    
6.  局限：仅适配静态图像数据，无法处理时序、文本类动态数据。
    

### 2\. 循环神经网络 RNN（专攻文本 / 时序数据）

1.  **文本数字化方案**
    
    -   单数字编码：缺陷多，无语义关联
        
    -   One-hot 编码：维度极高，无法表达词语关系
        
    -   **词嵌入**：最优方案，将文字转为低维稠密词向量，可通过向量相似度判断词语语义关联
        
2.  **RNN 核心特点**：逐词依次输入数据，保存前文隐藏状态，**捕捉文本先后顺序与时序逻辑**。
    
3.  适用场景：文本分类、语句翻译、文字生成。
    
4.  原生缺陷：长期依赖信息易丢失，必须串行计算，无法并行运算。
    

### 3\. Transformer 架构（NLP 主流核心架构）

1.  解决痛点：攻克 RNN 长期依赖丢失、串行低效两大问题。
    
2.  基础前置：词嵌入 +**位置编码**，给词向量添加语句先后位置信息。
    
3.  核心机制：**自注意力机制**
    
    -   生成 Q、K、V 三组向量矩阵，计算词语间相似度
        
    -   让每个词语直接关联整句所有上下文词语，全局捕捉语义关系
        
4.  **多头注意力**：多组 QKV 并行计算，提升语义学习灵活性与模型表达能力。
    
5.  整体结构：编码器 + 解码器组合，是大语言模型、AI 大模型的底层核心架构。
