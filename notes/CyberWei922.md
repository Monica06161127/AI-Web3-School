# 2026-05-20

今天学习了AI相关的内容，包括LLM，API等，LLM 并非数据库，而是一个非确定性的高维“世界模型”，其参数空间需要通过 Prompt 这一高维方向盘进行边界约束与上下文激活。为了将 LLM 的概率混沌转化为工业级的稳定输出，我们需要用 Workflow 构筑确定性的刚性骨架，并利用 Tool Use（函数调用） 赋予其连接外部 API 与环境的感知行动义肢。当 LLM 融合了记忆、规划与工具，便进化为具备自适应闭环决策能力的 Agent。这一技术组合在 AI Coding 领域展现出终极战力，代码编写被大模型彻底拉平，开发者的核心竞争力正从“熟练掌握语法”转向“定义问题与架构审美”。未来的系统不再是被逐行写出的，而是由人类指挥官调度手握 Tools 的 Agent，在 Workflow 的规范下共同编织出来的。
