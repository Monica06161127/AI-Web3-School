# **區塊鏈基礎概念**

區塊鍊：以區塊區塊串接的方式儲存所有資料，所有資料分布在世界各地，透明的交易狀態、資訊，構成去中心化的形式。  
壞處是，依但丟失就找不回來了

Web3 vs Web 3.0 vs Web2

-   Web2 現在用的google、FB、tiktok、amazon\\
    
-   Web3 去中心化互聯網  
    也是區塊鍊的一種  
    屬於自己的數位資產
    
-   Web3.0 語意網路  
    與Web3 可結合（語意標記+ 區塊鏈儲存）  
    **核心特徵：**
    
    -   **語意標記**：使用RDF、OWL 等標準描述資料關係
        
    -   **結構化資料**：資訊依照標準格式組織，方便機器理解
        
    -   **知識圖譜**：建構實體間的語意關係網絡
        
    -   **典型技術**：本體工程、語意查詢語言（SPARQL）、連結數據
